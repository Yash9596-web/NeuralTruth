# Tool Justification

| Tool | Justification |
|---|---|
| **DistilRoBERTa** | 40% smaller than BERT, retains 97% performance. Enables low-latency inference (<200ms). |
| **HuggingFace Transformers** | Industry-standard library for BERT fine-tuning and tokenization. |
| **FastAPI** | Async-first, OpenAPI-native Python REST framework — ideal for real-time ML inference APIs. |
| **PostgreSQL** | Production-grade relational DB for storing analysis history and user data. |
| **Redis** | In-memory caching layer for frequently queried domain scores and repeated claims. |
| **Next.js** | React framework with SSR/SSG for SEO-optimized, performant frontend. |
| **Docker** | Containerized deployment ensures reproducibility across all environments. |
| **Brave Search API** | Real-time web search for claim verification against trusted news sources. |
